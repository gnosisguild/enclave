# Enclave Examplee Project

This project demonstrates a basic Enclave use case.

## Getting started

### Initialize a project in an empty folder

```
enclave init
```

### Run a hardhat node

```
pnpx hardhat node
```

### Run a local ciphernode cluster

Run a local ciphernode cluster.

```
enclave nodes up
```

### Run the app in `dev` mode

```
pnpm dev
```
